## 关键决策点

| 位置 | 条件 | YES | NO |
|------|------|-----|-----|
| Step 3 | should_use_agent() | MarketRegime Agent | 算法兜底 |
| Step 4 | use_llm == True | 选股会议 | 算法选股 |
| Step 6 | is_profit == False | 触发优化 | 保持参数 |
| Step 8 | 满足固化条件 | 策略固化 | 继续训练 |

## 数据结构流转

```
输入参数
    │
    ▼
cutoff_date ──▶ RealDataLoader.load_all_data_before()
    │                  │
    │                  ▼
    │            stock_data: Dict[code -> DataFrame]
    │                  │
    │                  ▼
    │            compute_market_stats()
    │                  │
    │                  ▼
    │            market_stats ──▶ MarketRegime Agent
    │                  │                    │
    │                  ▼                    ▼
    │            regime: Dict{regime, confidence, params}
    │                  │
    │                  ▼
    │            SelectionMeeting / Selector
    │                  │
    │                  ▼
    │            selected: List[str]
    │                  │
    │                  ▼
    │            make_simple_plan() / Meeting
    │                  │
    │                  ▼
    │            TradingPlan
    │                  │
    │                  ▼
    │            SimulatedTrader.run_simulation()
    │                  │
    │                  ▼
    │            result: {return_pct, trades, ...}
    │                  │
    │                  ▼

    │                  │
    │                  ▼
    │            new_params: Dict
    │                  │
    │                  ▼
    └─────────────────▶ 汇总导出
```

## Agent 触发频率

| Cycle | 触发频率 | 说明 |
|-------|---------|------|
| 1 | 100% | 第1轮建立基线 |
| 30, 60, 90... | 3.3% | 每30轮触发一次 |
| 亏损后 | 100% | 连续亏损≥2次触发 |
| 市场变化 | 100% | Regime变化时触发 |

平均 LLM 调用率: ~10-15%
