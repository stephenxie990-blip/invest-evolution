# Archive Documentation

This directory stores the smallest historical set still needed by active
automation or audit-facing entrypoints.

Rules:
- Keep only documents that are still directly referenced by scripts or release workflows.
- Remove dated execution plans, handoff notes, and one-off remediation memos after closeout.
- Do not add active runbooks here; place active docs under `docs/`.
- Do not store runtime artifacts or generated outputs in this directory.
