# 历史归档区说明

`历史归档区/` 用于存放已经退出当前主链、但仍需要保留以便追溯、对比或参考的历史材料。

## 当前内容

- `历史归档区/planning/`：历史规划文件，如 `findings.md`、`progress.md`、`task_plan.md`
- `历史归档区/references/`：参考项目、知识库等外部资料
- `历史归档区/src_backup_from_old/`：旧源码备份
- `历史归档区/local_tooling/`：本地工具痕迹，如 `.learnings`、`.trae`
- `历史归档区/*.tar.gz`：历史打包归档

## 管理原则

- 归档内容不参与当前系统运行
- 原则上不再作为主链开发输入
- 如需引用历史资料，应优先在文档中说明用途，避免再把内容搬回根目录
